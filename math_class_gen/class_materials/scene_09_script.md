### 场景9课堂剧本

#### 场景定位
- 悟空与沙僧纠偏。
- 课堂空间保持不变，核心从八戒抢答转向学生之间的逻辑纠错。
- 镜头为中景对话镜头，保证五人都在同一课堂空间内可识别。

#### 人物调度
- 悟空位于左侧靠窗位置，是主要发言者之一。
- 沙僧在中间偏右接应补充，形成第二个理性声音。
- 八戒仍在中间偏左作前景反应；唐僧在讲台区域认可纠偏；小白龙在右侧认真听讲并记录。

#### 黑板内容
- 在八戒公式旁加批注：`本题不宜直接用夹角公式`。
- 新增思路箭头：`已知向量关系 -> 选取基底 -> 表示未知向量`。
- 保留原题，进入“方法选择”阶段。

#### 课堂剧本
镜头维持正面中景，先承接八戒刚才的发言，再把重心拉到左侧靠窗的悟空。悟空没有离开自己的座位，只是微微前探，人物旁有对话气泡，内含台词：“信息不足不能硬套公式”，把讨论从直觉套公式拉回到题目条件本身。

紧接着，中间偏右的沙僧稳稳接上，顺着黑板上的已知向量关系补充课堂逻辑，强调应该先选方法、再做表示。中间偏左的八戒听到这里，神情由自信逐渐转为疑惑，形成明显反差。

唐僧在讲台区域对这一纠偏方向表示认可，小白龙则在右侧靠门座位认真记录。黑板上新增的“本题不宜直接用夹角公式”与思路箭头，把学生之间的互动正式转化为课堂方法论，为师傅下一场系统授法铺路。
